# 概述

道易程是 DAO/dApp 的通证经济行为的基础治理层。

这意味着道易程是一个让 DAO暨dApp 繁衍生息、蓬勃发展的公共平台。

道易程的出现标志着我们迈入了元宇宙元器件时代！
